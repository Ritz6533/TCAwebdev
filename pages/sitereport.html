My Website Project

I have to say that making a website from the scratch is not that hard. I started to make this website through a rough sketch outlay of how my pages are going to look and gradually I modified it to make it better. I used an animation color gradient and the use of padding and alignment to make my website look better. And writing the codes along with checking the validation of the codes.

Color Selection

I am always fond of bw classic colors so it was not hard for me to go with the black and white style. Then I needed to choose a secondary color for my theme so I chose gold to match the black. for sections and to differentiate different parts blocks of the website I chose light and dark grey.

Website Design and Layouts

I had little knowledge about HTML CSS before studying this course. While working on the core part of the website I started to grasp the understanding of the fundamental layout and how the website is organized together to make an awesome website. A curve on the box can give an edge to its look and how it can make a difference from looking bulge to attractive design.

I have worked with WordPress and it was easy to choose a round box instead of a flat one with the help of different them builder however after learning core part of the web development I get to know how every pixel every part of the website every measurements works and make a difference in displaying the output to make an amazing website.

HTML and CSS Validator

During the process, I used HTML and CSS validator to check my codes. Time and again I checked it to make sure that the codes I write is following the guidelines. Compared to online drag and drop it is way difficult however I didn't feel that hard in the end. The more I tried to do more I realize how the part how the padding margin and how the CSS style works.
